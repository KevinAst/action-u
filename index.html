Introduction

action-u

??? RETROFIT THIS ... ver WowZee 789

The action-u library promotes several redux reducer composition utilities, which blend multiple reducers together forming a richer abstraction through functional decomposition (i.e. higher-order functions).

Reducer composition is not new. Redux itself provides the innovative combineReducers utility which allows you to fuse individual reducers together to build up the overall shape of your application state.

The most prevalent action-u utility is reducerHash(), which lets you combine sub-reducers in such a way as to eliminate the switch statement commonly used to delineate action type.

Additionally, action-u promotes other reducer compositions that can be used in conjunction with one another.
